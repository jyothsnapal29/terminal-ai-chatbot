# Terminal AI Chatbot (Python)
A modular terminal-based AI chatbot built with Python.

## Features

- Terminal chat interface
- Conversation memory
- Modular architecture
- Chat logging
- Mock AI responses for development

## Project Structure

src/
main.py
chat.py
utils.py
prompts.py

logs/
chat_log.txt

## Run Project
python src/main.py
